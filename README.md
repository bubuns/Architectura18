# Задание 2: Работа с Git

## Описание
Создание локального репозитория Git с несколькими коммитами и загрузка на GitHub.

## Структура проекта

- `app.py` - основной файл приложения
- `config.py` - конфигурационный файл
- `utils.py` - вспомогательные утилиты

## Инструкции по созданию репозитория

### 1. Инициализация локального репозитория

```bash
git init
```

### 2. Создание коммитов

```bash
# Первый коммит - базовая структура
git add app.py README.md .gitignore
git commit -m "Initial commit: базовая структура проекта"

# Второй коммит - добавление конфигурации
git add config.py
git commit -m "Add config: добавлен файл конфигурации"

# Третий коммит - добавление утилит
git add utils.py
git commit -m "Add utils: добавлены вспомогательные функции"
```

### 3. Создание удалённого репозитория на GitHub

1. Создайте новый репозиторий на GitHub через веб-интерфейс
2. Не добавляйте README, .gitignore или лицензию (они уже есть локально)

### 4. Подключение удалённого репозитория и отправка

```bash
git remote add origin https://github.com/ВАШ_USERNAME/ВАШ_РЕПОЗИТОРИЙ.git
git branch -M main
git push -u origin main
```

## Проверка

После выполнения команд вы сможете увидеть все коммиты в истории:

```bash
git log --oneline
```
